Simple To-Do App

A simple, lightweight To-Do application built with HTML, CSS, and JavaScript. Users can add, mark as complete, and delete tasks.

Features

Add new tasks.

Mark tasks as completed.

Delete tasks.

Responsive and minimal design.

Technologies Used

HTML – for the structure.

CSS – for styling.

JavaScript – for functionality.

Project Structure
simple-todo-app/
│
├─ index.html        # Main HTML file
├─ style.css         # Styling for the app
└─ script.js         # JavaScript functionality

Usage

Clone the repository or download the files.

Open index.html in your browser.

Start adding your tasks in the input field.

Click Add or press Enter to add a task.

Click on a task to mark it as complete.

Click the delete icon/button to remove a task.
